    Year Groups

    These functions are for use with year groups.

    Clean Year Group

    This function is designed to standardise the way that year groups appear in your data. Year Groups (1-13) are formatted to: 'Year i' or 'Reception'. It will also raise an error if a year group is not within the range 0 (Reception) to 13.

    Warning

    This function does not work on FE Levels and will raise an error. It will also raise an error if a year group is not within the range 0 (Reception) to 13. You can set error behaviour with the errors argument. 

    import heat_helper as hh

year_group_messy = ['Year 10', 'Y9', 10, 'year 11', 'reception', 'year r']

clean_year_groups = [hh.clean_year_group(yg, errors='ignore') for yg in year_group_messy]

print(clean_year_groups)

# Output: ['Year 10', 'Year 9', 'Year 10', 'Year 11', 'Reception', 'Reception']

    import heat_helper as hh
import pandas as pd

year_group_messy = ['Year 10', 'Y9', 10, 'year 11', 'reception', 'year r']

yg_dict = {'Year Group' : year_group_messy}

yg_df = pd.DataFrame(yg_dict)

yg_df['Clean YG'] = yg_df['Year Group'].apply(hh.clean_year_group)

print(yg_df.head(6))

# Output
#  Year Group   Clean YG
#0    Year 10    Year 10
#1         Y9     Year 9
#2         10    Year 10
#3    year 11    Year 11
#4  reception  Reception
#5     year r  Reception

    Calculate Year Group from Date

    This function calculates which year group a date of birth falls within. Year groups are calculated for the current academic year by default, but can be overridden to calculate year groups for past academic years. Year groups are returned in the format 'Year i' except for Reception which is returned as 'Reception'. If the function is calculates that the student is not yet old enough for school it will return 'Student too young for school.'

    Warning

    If returned year group is above 13, an error will be raised. If errors are ignored, None is returned. You can set error behaviour using the errors argument.

    import heat_helper as hh
from datetime import date

date_list = [date(2013,9,1), 
            date(2010, 5, 21), 
            date(2009, 10, 25), 
            date(2012, 10, 25)]

for dob in date_list:
    yg = hh.calculate_year_group_from_date(dob)
    print(f"Student with date of birth: {dob} is in: {yg}")

# Output
# Student with date of birth: 2013-09-01 is in: Year 7
# Student with date of birth: 2010-05-21 is in: Year 11
# Student with date of birth: 2009-10-25 is in: Year 11
# Student with date of birth: 2012-10-25 is in: Year 8

    import heat_helper as hh
import pandas as pd
from datetime import date

dob_dict = {'DOB' : [date(2013,9,1), 
                    date(2010, 5, 21), 
                    date(2009, 10, 25), 
                    date(2012, 10, 25)]}

date_df = pd.DataFrame(dob_dict)
date_df['DOB'] = pd.to_datetime(date_df['DOB']).dt.normalize()

date_df['Year Group'] = date_df['DOB']
                        .apply(hh.calculate_year_group_from_date)

print(date_df)

# Output
#         DOB Year Group
#0 2013-09-01     Year 7
#1 2010-05-21    Year 11
#2 2009-10-25    Year 11
#3 2012-10-25     Year 8

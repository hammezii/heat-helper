# Getting Help
If you are new to `heat_helper`, want to find out about its functions, or see examples of it in use, the following pages can help.

## :material-progress-question: Usage Documentation
The **[usage documentation](names.md)** contains an explanation of how each function works and examples of it in use. 

## :material-api: API Documentation
The **[API documentation](names-doc.md)** contains detailed information on the arguments of each function and datatypes it expects, as well as possible errors it can raise.

## :material-github: GitHub 
Something not working as expected? You can [raise an issue](https://github.com/hammezii/heat-helper/issues) on GitHub.
